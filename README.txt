Compile & Run Instructions

You can run the program using eclipse IDE or the instructions below.

===============================================================================
To run the program in commandline:
-------------------------------------------------------------------------------
Linux:
	in ../project/project> javac -d bin -cp bin src/launcher/Main.java
	in ../project/project> java -cp .:bin:/*.class launcher.Main 
-------------------------------------------------------------------------------
Windows:
	in ../project/project> javac -d bin -cp bin src\launcher\Main.java
	in ../project/project> java -cp .;bin;\*.class launcher.Main 
===============================================================================
